Cómo funciona SonarIA

Un flujo simple, rápido y pensado para docentes: ingresas una canción y recibes una orientación inicial útil para decidir su uso pedagógico.

1
Ingresa título y artista

Puedes además pegar la letra completa o dejar que SonarIA intente buscarla automáticamente.

2
SonarIA analiza

Detecta lenguaje sensible, señales emocionales y posibles riesgos para uso escolar.

3
Recibes orientación

Obtienes resultado, nivel de riesgo, temas detectados y una justificación inicial para tomar decisiones.
